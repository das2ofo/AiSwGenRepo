◆ TempMon_Run()

void TempMon_Run ( void )

Monitor ECU temperature and update the over-temperature diagnostic state.

This function:

  • Reads the filtered ECU temperature from TempMeas_get_filtEcuTemp_s16().
  • Updates the debounced over-temperature flag TempMon_OverTempFlag_b.
  • Updates the debounce counter count_u8.
  • Updates the diagnostic state EC_Set_error_overtemperature_State_e according to the flag transitions.

The formal behaviour of this function is specified by the ACSL contract immediately following this Doxygen comment and can be checked with Frama-C (WP + E-ACSL).
